    Ark: Survival Evolved

    MMO Survival

    We host 12 cross Ark servers that feature no mods and a log of benefits. These benefits include vote rewards, login points, an in game shop, and weekly events that include many different prizes.

    Ark: Survival Ascended

    MMO Survival

    Ark: Survival Evolved remastered on Unreal Engine 5. We host all the servers for you to join. Benefits also includ vote rewards, login points, and an ingame shop.

    Eco: Global Survival

    Sandbox Simulation

    Eco is a sandbox simulation game where you work together to destroy a meteor hurling at the planet. We host a 30 day rotating wipe of Eco.

    Minecraft: Java Edition

    Sandbox

    We host survival, creative, bedwars, parkour, and many other servers all connected together. You can also earn points and vote rewards on our servers. There is also an ingame shop.

    Empyrion: Galactic Survival

    Space Survival

    Empyrion is our longest ran server with a loyal playerbase. This game features the Reforged Eden Playthrough offering a near infinite universe to explore.
